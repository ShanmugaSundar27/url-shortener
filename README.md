# url-shortener

A tiny URL shortener service — POST a URL to get a short ID, then visit `/r/:id` to redirect to the original.

## Quick Start

### Prerequisites

- Node.js >= 20
- npm (comes with Node.js)

### Setup

```bash
git clone https://github.com/ShanmugaSundar27/url-shortener.git
cd url-shortener
npm install
```

### Run

```bash
npm run dev
```

The app will be available at <http://localhost:3000>
